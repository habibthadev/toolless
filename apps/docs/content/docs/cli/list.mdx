---
title: toolless list
description: List databases or collections in your Toolless project
---

The `list` command displays databases or collections with their document counts and sizes.

## Usage

```bash title="Terminal"
toollessdb list [database] [options]
```

## Alias

```bash title="Terminal"
toollessdb ls [database] [options]
```

## Examples

### List All Databases

```bash title="Terminal"
toollessdb list
```

Output:

```
┌──────────┬─────────────┬───────┐
│ Database │ Collections │ Size  │
├──────────┼─────────────┼───────┤
│ myapp    │ 3           │ 45 KB │
│ testdb   │ 2           │ 12 KB │
└──────────┴─────────────┴───────┘
```

### List Collections in a Database

```bash title="Terminal"
toollessdb list myapp
```

Output:

```
┌────────────┬───────────┬────────┐
│ Collection │ Documents │ Size   │
├────────────┼───────────┼────────┤
│ users      │ 150       │ 23 KB  │
│ posts      │ 420       │ 18 KB  │
│ comments   │ 1.2K      │ 4.1 KB │
└────────────┴───────────┴────────┘
```

### List from Custom Path

```bash title="Terminal"
toollessdb list -p /path/to/databases
```

## Notes

- Database sizes include all collection files and metadata
- Document counts are fetched in real-time
- File sizes are displayed in human-readable format (KB, MB, GB)
- Hidden collections (prefixed with `_`) are excluded from the list
